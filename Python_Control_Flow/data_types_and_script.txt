a. {i} The Various Data Types in Python.

(-) 'int' in Python refers to a data type of whole numbers. Examples; '2', '4', '5'.

(-) 'float' in Python refers to a data type of decimals. Examples; '4.5', '6.7', '3.9'.

(-) 'str' in Python refers to a data type of words. Examples; 'school', 'banana', 'Teshie'.

(-) 'list' in Python refers to a data type of a collection of items. Examples; ('lion', 'elephant', 'giraffe', 'rabbit')
